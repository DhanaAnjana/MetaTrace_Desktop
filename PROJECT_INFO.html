Project Purpose & Privacy
Aspect Details
Purpose Digital forensics and cybersecurity investigation support. Designed to help security professionals analyze metadata from multiple files and identify connections, patterns, and potential evidence of malicious activity or file tampering.
Data Privacy All processing is local. No files or metadata are sent to external servers without explicit user action. Email functionality uses personal Gmail credentials with app passwords (secure method).
File Handling Files are analyzed but never modified. Hashes are calculated for integrity verification. File signatures are validated locally against known formats.
Logs & Evidence All activities are logged with timestamps and SHA256 integrity hashes. Logs are stored locally and can be exported for case documentation.
How to Use MetaTrace
1. Select Files: Click "Select Files" to choose individual files or "Scan Folder" for batch analysis.

2. Analyze: Click "Analyze" to extract metadata and find correlations automatically.

3. Review Results: View extracted metadata, anomalies, and correlations in the tree display.

4. Export Reports: Export results as JSON (data), CSV (spreadsheet), PDF (professional), or HTML (interactive).

5. Email Reports: Send generated reports via Gmail using app passwords (secure method).

6. View Logs: Check activity logs with timestamp and integrity verification.
